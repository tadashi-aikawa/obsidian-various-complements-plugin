- Requirements
    - Node.js v22

### Development

#### Set up

```bash
git config core.hooksPath hooks
```

#### Install dependencies

```bash
corepack enable
pnpm install
```

#### Build for development

```bash
pnpm dev
```

#### Test

```bash
pnpm test
# or
pnpm test --watch
```

#### CI

```bash
pnpm run ci
```

#### Release

Run [Release Action](https://github.com/tadashi-aikawa/obsidian-various-complements-plugin/actions/workflows/release.yaml).
